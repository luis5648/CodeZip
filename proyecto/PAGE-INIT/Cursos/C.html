Lenguaje C

Curso de C.En este aprenderás todo lo básico acerca de este lenguaje, podrás ir realizando ejercicios y tomar lecciones de teoría de las cuales todos nuestros tutores disponibles se encargarán de ir actualizando así como de calificar tu aprendizaje. como ejmplo de este curso básico podría adelantarse que podrás aprender desde como crear una variable hasta como resolver ecuaciones matematicas con lo aprendido. Así mismo contaremos con otros temas importartes dentro de C para personas que vayan más avanzados.

Programación Paralela

Con el paso del tiempo lograremos realizar algoritmos más complejos, por lo tanto ocuparemos de un mayor procesamiento en nuestros computadores es aquí donde entra la programación paralela que es donde aprenderos a utilizar todos los nucleos de nuestro cpu para así ayudar a que nuestros algoritmos sean resuelto de una manera mas rápida y eficaz.

Programación de medio nivel

Una vez entendidos y dominado los anteriores, podremos pasar al siguiente nivel en el cúal podremos aprender el uso de listas ligadas, tablas hash, así como algoritmos de busqueda complejos.
